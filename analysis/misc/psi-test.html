Este documento documenta el resultado de usar solve_poisson() para obtener la funcin corriente (en particular, su anomalas zonal) a partir de la vorticidad.

Dado que ERA provee vorticidad pero no funcin corriente, tena que usar NCEP. Pero NCEP tiene funcin corriente en (algunos) niveles sigma, por lo cual si yo hago el anlisis usando ERA5 en niveles de presin, la comparacin con la funcin corriente tiene ya dos incompatibilidades (aunque tampoco tan grandes, preliminarmente, los resultados no varan mucho usando la funcin corriente de NCEP en sigma 0.2101 o la funcin corriente derivada de ERA5 en 200 hPa.).

En cualquier caso, antes de usar solve_poisson() quiero chequear que funciona correctamente. Lo que voy a hacer es usar la vorticidad de NCEP en sigma 0.2101 para obtener la funcin corriente y comparar el resultado con el valor descargado de NCEP.

Como detalle, si a la funcin corriente se le suma y multiplica constantes, sigue siendo solucin de la ecuacin de Poisson, por lo que la solucin obtenida por solve_poisson() puede tener cualquier escala que nada que ver. Para resolver esto, voy a escalar la funcin corriente derivada con constantes obtenidas a partir de hacer lm(psi ~ ps_derivada).

psi <- NCEP_PSI() %>% 
  ReadNetCDF(subset = list(level =  0.2101, 
                           time = c("1979-01-01", NA))) %>% 
  rm_singleton()

vor <-  NCEP_VOR() %>% 
  ReadNetCDF(subset = list(level =  0.2101, 
                           time = c("1979-01-01", NA))) %>% 
  rm_singleton()


ncep <- vor[psi, on = .NATURAL] %>% 
  na.omit()
rm(psi, vor)
psi_derived <- ncep[, solve_poisson(vor, lon, lat), by = time]
setnames(psi_derived, "value", "psi_derived")
ncep <- ncep[psi_derived, on = .NATURAL]


scaling <- ncep[, FitLm(psi, psi_derived)]
ncep[, psi_derived := psi_derived*scaling$estimate[2] + scaling$estimate[1]]
ncep[, `:=`(psi         = Anomaly(psi),
           psi_derived = Anomaly(psi_derived)), by = .(lat, time)]

La Figura 1 muestra cuatro campos de anomala zonal de geopotencial tomados al azar. La funcin corriente derivada con es casi indistinguible de la funcin corriente real. La correlacin global durante todo el perodo entre la funcin corriente real y la derivada es de 0.9993348. Prcticamente perfecto.
